Intent

Create desktop shortcuts which will always run CMD/PowerShell with admin rights and change to some directory.

We may want to run CMD and Powershell consoles with Admin rights to access virtual hosts: myvm1 and myvm2

Procedure

Create a new Folder on the desktop - "Docker Commands" and copy shortcuts, for example:

Make sure they all run with Admin rights.

From “Properties | Advanced”

Set starting directory

Set “Target” from “Properties”

Target:

Value:

CMD:

%windir%\system32\cmd.exe /k cd %userprofile%\Docker

PowerShell:

%SystemRoot%\system32\WindowsPowerShell\v1.0\powershell.exe -NoExit -command "& {Set-Location $env:userprofile\Docker}"
