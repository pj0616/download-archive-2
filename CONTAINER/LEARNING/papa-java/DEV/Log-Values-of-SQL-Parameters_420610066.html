DevOps : Log Values of SQL Parameters

Created by Nicholas Guner on Jun 05, 2020

Refer to:

URL

https://stackoverflow.com/questions/1710476/how-to-print-a-query-string-with-parameter-values-when-using-hibernate


When we insert data, log output looks like this:

Hibernate: insert into project (created_by, in_utc, last_modified_by, out_utc, audit_id, project_description, project_name, id) values (?, ?, ?, ?, ?, ?, ?, ?)

This is not helpful if we need to diagnose a problem.

We can append following likes to application.properties file:

...
logging.level.org.hibernate.SQL=DEBUG
logging.level.org.hibernate.type=TRACE

Now, the output looks like this:
