DevOps : Windows Batch Commands

Created by Nicholas Guner on Jan 29, 2021

Rename all files recursively from current directory

Command:

for /r . %i in (*.js) do ren "%i" *.nick

Notes:

This will rename all files with *.js extension to *.nick extension.
