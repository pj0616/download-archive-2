DevOps : Use ObjectMapper instead of JSONObject in integration tests when controllers return List<?>

Created by Nicholas Guner on Jul 02, 2020

Refer to following URL for other ways of accomplishing this task:

URL

https://stackoverflow.com/questions/6349421/how-to-use-jackson-to-deserialise-an-array-of-objects


Many of SoD controllers return a list of Entity of DTO objects, for example:

@GetMapping(value = "/assignables")
public ResponseEntity<List<AssignableDTO>> getAssignables()
{...}

When we initially wrote integration tests for these controller APIs, we resorted to JSONObject to retrieve fields of DTO object, for example:

@Test
public void testA_getAllAssignables() throws JSONException
{
    //  Verify existence of all assignables
    Set<Integer> assignableIDs = IntStream.rangeClosed(1, 34).boxed().collect(Collectors.toSet());
    ResponseEntity<String> response = testRestTemplate.getForEntity(createUrlWithPort( "/sod/assignables"), String.class);
    assertThat(response.getStatusCode(), equalTo(HttpStatus.OK));
    JSONArray jsonArray = new JSONArray(response.getBody());
    for (int i = 0; i < jsonArray.length(); ++i)
    {
        JSONObject jsonObject = (JSONObject) jsonArray.get(i);
        int id = (int)jsonObject.get("ID");
        assignableIDs.remove(id);
    }
    assertTrue(assignableIDs.isEmpty());
}

This approach relies on correctly specifying the name of the field. We would ultimately like to case result to a List<AssignableDTO> objects and call methods on theÂ AssignableDTO class. This approach is possible with ObjectMapper, for example:

@Test
public void testA_getAllAssignables() throws IOException
{
    //  Verify existence of all assignables
    List<Long> expectedAssignableIDs = LongStream.rangeClosed(1, 34).boxed().collect(Collectors.toList());
    ResponseEntity<String> response = testRestTemplate.getForEntity(createUrlWithPort( "/sod/assignables"), String.class);
    assertThat(response.getStatusCode(), equalTo(HttpStatus.OK));
    String content = response.getBody();
    ObjectMapper objectMapper = new ObjectMapper();
    List<AssignableDTO> assignableDTOs = objectMapper.readValue(content, new TypeReference<List<AssignableDTO>>(){});
    List<Long> actualAssignableIDs = assignableDTOs.stream().map(AssignableDTO::getID).sorted().collect(Collectors.toList());
    assertArrayEquals(expectedAssignableIDs.toArray(new Long[0]), actualAssignableIDs.toArray(new Long[0]));
}
