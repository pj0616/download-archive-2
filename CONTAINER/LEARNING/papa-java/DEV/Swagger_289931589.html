Create Model, Server, and Controller Classes

Modify pom.xml to add Swagger Dependencies

<dependency>
<groupId>io.springfox</groupId>
<artifactId>springfox-swagger2</artifactId>
<version>2.9.2</version>
</dependency>

Add Configuration for Swagger Docket Bean

Create a new package under root:

Package

  • config

Add Docket bean for Swagger inside the config package

@Configuration
@EnableSwagger2
public class SwaggerConfig {
 @Bean
 public Docket api() {
        return new Docket(DocumentationType.SWAGGER_2)
                .select()
                .apis(RequestHandlerSelectors.any())
                .paths(PathSelectors.any())
                .build();
    }
}

Verify that SpringFox is working

Launch the application and point your browser to:

URL

http://localhost:8080/v2/api-docs

You should see some text being displayed, for example:

Output:

{"swagger":"2.0","info":{"description":"Api Documentation","version":"1.0","title":"Api Documentation" ...

Add Swagger UI

Modify pom.xml to add. Note that later version, such as 2.9.2 may not produce desired output.

<dependency>
<groupId>io.springfox</groupId>
<artifactId>springfox-swagger-ui</artifactId>
<version>2.6.1</version>
</dependency>

Restart application and point your browser to:

URL:

http://localhost:8080/swagger-ui.html

which will expose all REST APIs from the Controller
