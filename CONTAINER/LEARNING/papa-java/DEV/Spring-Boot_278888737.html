Warning about Package Locations for Service and RestController Classes

While working with a Spring Boot application, you will most likely need to create a Service class (decorated with @Component annotation) and a Rest Controller class (decorated with a @RestController annotation). You may place your Service and the RestController in the same package as you Spring Boot Application class (the one that is decorated with @SpringBootApplication annotation) or in child packages. If you place your Service and RestContoller classes in packages that are at the same level as your SpringBootApplication class, Spring will not be able to find them during its scan.
