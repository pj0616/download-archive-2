Abstract Factory

Description

Abstract Factory is a super-factory which creates other factories. We can call it "Factory of factories".

Example

We want to build two types of computers: AMD and Intel. Want to make sure to use appropriate parts for each.

Client Code

public class Client
{
  public static void main(String[] args)
  {
    new Computer(createSpecificFactory(arg[0]));
  }
 
  public static CPUFactory createSpecificFactory(String type)
  {
    if("AMD".equals(type)) return new AMDFactory();
    else return new IntelFactory();
  }
}
class Computer
{
  CPU cpu;
  Motherboard mb;
 
  public Computer(CPUFactory factory)
  {
    cpu = factory.produceCPU();
    cpu.process();
    mb = factory.processMotherboard();
    mb.process();
  }
}
