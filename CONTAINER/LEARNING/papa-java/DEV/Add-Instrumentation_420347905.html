Adding Instrumentation

If Actuator was added to this project from SPRING INITIALIZR (https://start.spring.io/), the pom.xml should have the following dependency:

<dependency>
   <groupId>org.springframework.boot</groupId>
   <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

We will also need to add following dependency for the HAL Browser

<dependency>
   <groupId>org.springframework.data</groupId>
   <artifactId>spring-data-rest-hal-browser</artifactId>
</dependency>

Additionally, we need to add a property to application.properties file to expose management endpoints:

...
management.endpoints.web.exposure.include=*

After these changes, our application can display the HAL Browser:

URL

Display

http://localhost:8081/browser/index.html#/

We can view links by typing "actuator" in the Explorer field and clicking "Go!"

Which will expose following links


One of the links in "info" but it does not display anything useful:

We can add properties to application.properties file to display something useful:

...
management.endpoints.web.exposure.include=*
info.app.name=@project.name@
info.app.description=@project.description@
info.app.version=@project.version@
info.app.encoding=@project.build.sourceEncoding@
info.app.java.version=@java.version@
...

Note that values for these monikers will come from pom.xml file. Now, when we check the "Info" link, we will see following:
