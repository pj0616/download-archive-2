Create new DB2 Schema for storing data

See DB Portal for instructions to create new schema or use SOD_TEST.

We will opt for the SOD_TEST schema

Connect with promproxy account and drop all tables from SOD_TEST schema.

set schema SOD_TEST

drop table duty;
drop table duty_privilege_map;
drop table duty_rule_map;
drop table key_store;
drop table privilege;
drop table properties;
drop table rule;
drop table rule_properties_map;
