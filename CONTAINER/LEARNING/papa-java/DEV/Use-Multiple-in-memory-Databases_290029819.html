DevOps : Use Multiple in-memory Databases

Created by Nicholas Guner on Apr 19, 2020

Modify application.properties files to support two configurations

Create two new application.properties files:

resources/application-gunern.properties

spring.profiles.include=h2
spring.jpa.properties.hibernate.hbm2ddl.import_files=data-gunern.sql

resources/application-katzj.properties

spring.profiles.include=h2
spring.jpa.properties.hibernate.hbm2ddl.import_files=data-katzj.sql

Create two run configurations

Configuration Name

Program Arguments

gunern

  • -spring.profiles.active=gunern

katzj

  • -spring.profiles.active=katzj
