Arrays

Check if array elements are all unique. Note use of Set and boolean array for checking. Usde of boolean array is interesting because it uses array index.

https://nickguner.atlassian.net/wiki/spaces/DEV/pages/858718211/Arrays+Problems#AllUnique

https://nickguner.atlassian.net/wiki/spaces/DEV/pages/858718211/Arrays+Problems#BinarySearch

Count Sum. Make sure you understand CountSum for two elements and CountSum for four (or three elements). Note that the array must be sorted.

https://nickguner.atlassian.net/wiki/spaces/DEV/pages/858718211/Arrays+Problems#CountSum

https://nickguner.atlassian.net/wiki/spaces/DEV/pages/858718211/Arrays+Problems#Fibonacci
