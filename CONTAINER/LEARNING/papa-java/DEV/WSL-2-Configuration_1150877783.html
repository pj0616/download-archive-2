Bash to Windows Directories

Bash

/mnt/c/Users/nick

Windows

%userprofile% (C:\Users\nick)

Share files between Windows and Linux

Create a “shared” directory in Windows, for example, %userprofile%\WslFiles

In Ubuntu, create a soft link:

Command:

ln -s /mnt/c/Users/nick/WslFiles work

Note:

/mnt/c/Users/nick/WslFiles is the same location as /home/nick

Test

In Ubuntu:

Commands:

cd work

touch nick.xyz

In Windows, note the nick.xyz file in %userprofile%\WslFiles directory:

Create a .profile for Ubuntu

Change to the home directory (command: cd)

Rename existing .profile file:

Command:

mv .profile .profile.original

Create a new .profile file with vi. Note: you may need to execute :set paste for the text to paste without indents. Use shift+ins to insert text:

export ENV_OS=$(uname)
export HOSTNAME=`hostname -s`

PS1=$'\[\e[31m\]'`whoami`$'\[\e[0m\]@'$'\[\e[0;36m\]'$HOSTNAME$'\[\e[0m\]:'$'\[\e[32m\]'`pwd`$'\[\e[0m\]> '

mycd()
{
  unalias cd
  cd $1
  PS1=$'\[\e[31m\]'`whoami`$'\[\e[0m\]@'$'\[\e[0;36m\]'$HOSTNAME$'\[\e[0m\]:'$'\[\e[32m\]'`pwd`$'\[\e[0m\]> '
  alias cd=mycd
}

set -o vi

alias cd=mycd

Set up vi editor as default

For Git

Command:

git config --global core.editor "vim"

For v command from less

Command:

sudo update-alternatives --config editor
