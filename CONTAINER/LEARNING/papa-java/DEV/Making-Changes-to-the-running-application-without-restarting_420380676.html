DevOps : Making Changes to the running application without restarting

Created by Nicholas Guner on Jun 05, 2020

Add DevTools dependency to pom.xml

...

<dependency>
   <groupId>org.springframework.boot</groupId>
   <artifactId>spring-boot-devtools</artifactId>
   <scope>runtime</scope>
   <optional>true</optional>
</dependency>
...

Changes to Intellij to support Live Reloads

In IntelliJ, choose "Project Settings", "Build, Execution, Deployment", "Compiler" and check "Build project automatically"

In IntelleJ, press CTRL+SHFT+A and select "Registryâ€¦", check "compiler.automake.allow.when.app.running"

Note: "Restore defaults" when finished with this experiment.

Make a coding change in data.sql

FROM:

INSERT INTO PROJECT(ID, AUDIT_ID, IN_UTC, OUT_UTC, PROJECT_NAME, PROJECT_DESCRIPTION, REVENUE, CREATED_BY, LAST_MODIFIED_BY) VALUES(1, '00000000-0000-0000-0000-000000000001', CURRENT_TIMESTAMP, '9999-12-31', 'P-1', 'Project one', 1200000.00, 'unknown', 'unknown')

TO:

INSERT INTO PROJECT(ID, AUDIT_ID, IN_UTC, OUT_UTC, PROJECT_NAME, PROJECT_DESCRIPTION, REVENUE, CREATED_BY, LAST_MODIFIED_BY) VALUES(1, '00000000-0000-0000-0000-000000000001', CURRENT_TIMESTAMP, '9999-12-31', 'P-1', 'Project uno', 1200000.00, 'unknown', 'unknown')
