Use ticks (`)

Use Number, String, Boolean to convert types

prompt will always return String. Use Number to convert.

Number('abc') returns NaN (not a number)

“+” will always take precedence when one of the variables is a string

Falsy values

Values that are coursed to false:

0, '' (empty string), undefined, null, NaN

Use “===” instead of “==” for strict conversions

“===” avoids performing type conversion

‘18’ === 18

false

‘18’ == 18

true

switch uses strict equality

Activate Strict mode in JS

'use strict';

Object properties may be accessed with dot or brackets

When accessed with brackets, use String expressions

let first = 'first';

obj.firstNama
obj['firstName'], obj['first' + 'Name'], obj[first + 'Name']

Configure Snippets

Allows you to create macros. Select “User Snippets” from Settings and choose “New Global Snippets file…”. Give this file some name. Uncomment code. For console.log(), enable following section:

{
  "Print to console": {
    "scope": "javascript,typescript",
    "prefix": "cl",
    "body": ["console.log();"],
    "description": "Log output to console"
  }
}

Type “cl” in the editor and console.log(); will get printed.
